五子棋与围棋网页版
基于原生Web技术开发的轻量级棋类对弈平台，纯前端实现，无需后端依赖，浏览器打开即可运行五子棋与围棋双人对弈。
🎮项目介绍
本项目是一套简洁美观、交互友好的棋类游戏网页版，实现经典五子棋与围棋的完整双人对战能力，界面优雅、逻辑严谨，适合学习、娱乐与演示使用。
✨核心功能
双游戏模式支持：五子棋+围棋，自由切换
棋盘尺寸可选：支持8×8～19×19多种棋盘大小
智能棋盘渲染：DOM动态绘制棋盘线条与交叉点
精准落子逻辑：自动吸附交叉点，禁止重复落子
黑白轮流落子：回合提示清晰，操作直观
五子棋胜负判定：实时检测横、竖、左斜、右斜四方向，五连子即判胜
围棋完整规则：支持提子、禁着点、禁止自杀、连续虚着结束
围棋中国数子法：终局自动计算胜负、贴子规则准确
悔棋功能：双方各限3次，支持历史状态回退
游戏控制：重新开始、虚着、投子认负、返回主页
棋谱管理：支持保存棋谱到本地JSON文件，可加载复盘
操作提示面板：规则说明一键显示/隐藏
响应式界面：适配不同屏幕尺寸，视觉精致
🛠️技术栈
HTML5：页面结构、游戏容器与布局搭建
CSS3：渐变背景、阴影质感、弹性布局、界面美化
原生JavaScript：面向对象设计、游戏核心逻辑、事件处理、状态管理
设计模式：采用基类继承、备忘录模式实现状态保存与悔棋
🚀快速开始
下载或克隆项目源码
将三个文件放在同一目录下：index.html、style.css、script.js
直接用浏览器打开index.html即可运行
在首页选择棋盘尺寸，点击进入五子棋或围棋开始对战
📂文件结构
index.html项目主页面，包含首页、游戏页、提示面板
style.css全局样式文件，负责UI渲染与布局美化
script.js核心游戏逻辑，包含五子棋、围棋、UI管理
📌使用说明
五子棋：黑方先行，率先连成五子即可获胜
围棋：轮流落子，提掉无气棋子，禁止自杀与禁着点；连续两次虚着自动数子判胜
悔棋：每方最多悔棋3次
可保存当前对局到本地，随时加载继续对战
