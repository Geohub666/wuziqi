# 五子棋小游戏 - 课程作业
## 项目介绍
基于HTML5、CSS3、JavaScript开发的15x15标准五子棋，实现落子、胜负判断、重新开始等核心功能。

## 运行方法
1.  克隆仓库：`git clone https://github.com/你的GitHub用户名/wuziqi.git`
2.  直接用浏览器打开`index.html`即可开始游戏。

## 功能说明
- 黑白双方轮流落子
- 自动判断胜负（横/竖/斜向连成五子获胜）
- 重新开始游戏功能
