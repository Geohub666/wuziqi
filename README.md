五子棋与围棋网页版
本项目是一个基于Web技术开发的棋类对弈平台。通过简洁的界面，实现了经典的五子棋(Gomoku)和围棋(Go)的核心对弈功能。

🎮项目演示
●五子棋模式：支持黑白交替落子，自动判定五子连珠胜负。
●围棋模式：实现落子逻辑与棋盘交互。

✨核心功能
●棋盘渲染：利用HTML5 Canvas/DOM动态绘制15×15（五子棋）或19×19（围棋）的标准棋盘。
●落子逻辑：
○精确捕捉鼠标点击坐标，实现自动对齐交叉点。
○黑白棋子轮流落子，防止重复落子。
●胜负判定：
○五子棋：实时检测横、竖、左斜、右斜四个方向是否达成五连子。
●交互功能：
○重新开始：一键清空棋盘，重置游戏状态。
○落子音效/提示：辅助提升对弈体验（如已实现）。

🛠️技术栈
●HTML5：负责页面结构与棋盘画布搭建。
●CSS3：实现自适应布局、棋子阴影质感及界面美化。
●JavaScript(Vanilla JS)：
○处理核心游戏算法（胜负检测、逻辑判断）。
○DOM操作与事件监听。

🚀快速开始
1.克隆本项目或下载源代码。
2.在浏览器中直接打开index.html文件即可开始游戏。
3.通过界面顶部的切换按钮（如有）选择游戏类型。
